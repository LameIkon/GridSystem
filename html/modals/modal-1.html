Objective.

Reach the Goal by moving the Player each turn. Enter a number in the boxes below and press ㅤRunㅤ to execute the movement.

Theory.

This game features four core functions: MoveUp, MoveDown, MoveRight, and MoveLeft.
Each function contains the logic needed to move the player in the respective direction. To determine the number of steps the player moves, simply input a value into the function's parameter.
