Objective.

Reach the goal by moving the player each turn. You can write a number into the boxes below and press run.

Theory.

This game has 4 functions: MoveUp, MoveDown, MoveLeft and MoveRight. Each of them holds code that moves the player, but the functions must know how many steps to move, by inserting a number in the parameter.
